# Числа в JavaScript

## Бинарные числа

Синтаксис бинарных чисел использует ведущий ноль, за которым следует строчная или заглавная латинская буква "B" (0b или 0B).
Если цифры после 0b не являются 0 или 1, возникает следующая ошибка SyntaxError: "Missing binary digits after 0b".

## Как сложить бинарные числа? 

BigInt(b1) + BigInt(b2) + .toString(2) если нужно привести в двоичную систему


## Пример

```js
(25).toString(2) // '11001'

new BigInt('0b11001') + new BigInt('0b11001') // '50n'

(BigInt('0b11001') + BigInt('0b11001')).toString(10) // 50
```
