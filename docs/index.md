## Команды Cursor

В проекте используется команда **Process Task**: обрабатывает файл с алгоритмической задачей — переносит в нужную папку, очищает код, добавляет JSDoc/теги и пишет тесты.

**Как использовать:** Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`) → «Process Task». Можно передать путь к файлу в параметрах.

Подробнее: **[Как добавить задачу](/commands)**
