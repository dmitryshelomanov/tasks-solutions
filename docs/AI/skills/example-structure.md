# Пример структуры SKILL.md

Минимальный скилл состоит из YAML frontmatter и тела с инструкциями.

## Минимальный пример

```markdown
---
name: my-skill
description: Кратко описывает, что делает скилл и когда его применять. Use when the user asks for X or mentions Y.
---

# Название скилла

## Инструкции

1. Шаг первый — что сделать.
2. Шаг второй — как проверить результат.
3. Шаг третий — в каком формате выдать ответ.

## Пример

Пользователь: «Сделай X»

Агент:
- Выполняет шаг 1…
- Выполняет шаг 2…
- Возвращает результат в указанном формате.
```

## Обязательные поля во frontmatter

| Поле | Ограничения | Назначение |
|------|-------------|------------|
| `name` | До 64 символов, только латиница, цифры, дефисы | Уникальный идентификатор скилла |
| `description` | До 1024 символов, не пустое | По нему агент решает, применять ли скилл к запросу |

## Расширенная структура папки скилла

```
my-skill/
├── SKILL.md        # Обязательно — основные инструкции
├── reference.md    # Опционально — подробная справка
├── examples.md     # Опционально — дополнительные примеры
└── scripts/       # Опционально — вспомогательные скрипты
```

В `SKILL.md` можно ссылаться на эти файлы, чтобы агент подгружал детали по мере необходимости.
