# Основные структуры данных: Массивы, Векторы и Кортежи

## Массив

- **Определение**: Массив — это переменная, которая может хранить несколько значений одного типа.
- **Синтаксис**: Значения указываются в квадратных скобках `[]`.
- **Особенности**:
  - Размер массива фиксирован (в некоторых языках программирования, таких как Rust).
  - Все элементы массива должны быть одного типа.
- **Пример**:

```rust
let numbers = [1, 2, 3, 4, 5]; // Массив целых чисел
```

## Вектор

- Определение : Вектор — это динамический массив, который может увеличиваться или уменьшаться в размере.
- Синтаксис : Создаются с помощью метода vec! в Rust.

### Особенности :

- Вектор можно рассматривать как одномерный список данных.
- Размер вектора не фиксирован, он может расширяться по мере добавления новых элементов.

```rust
let mut numbers = vec![1, 2, 3];
numbers.push(4); // Добавление элемента
```

## Кортеж

- Определение : Кортеж — это коллекция значений, которые могут быть разных типов.
- Синтаксис : Элементы кортежа указываются в круглых скобках ().

### Особенности:

- Кортежи позволяют хранить значения разных типов в одной структуре.
- Размер кортежа фиксирован, но каждый элемент может иметь свой тип.

```rust
let person = ("Alice", 30, true); // Кортеж из строки, числа и булевого значения
```

## Сравнение массива и вектора

### Массив:

- Фиксированный размер.
- Хранит значения одного типа.
- Подходит для ситуаций, когда количество элементов известно заранее.

### Вектор:

- Динамический размер.
- Также хранит значения одного типа.
- Подходит для ситуаций, когда количество элементов может меняться.

## Аналогия

- Вектор можно представить как одномерный список данных. Это "строка" данных.
- Массив — это список, организованный в несколько измерений. Например, двумерный массив — это вектор векторов одинаковой длины.

## Примечания

- Векторы и массивы часто используются для хранения последовательностей данных, но выбор между ними зависит от требований к изменяемости размера.
- Кортежи полезны, когда нужно объединить разнородные данные в одну структуру, например, для возврата нескольких значений из функции.
