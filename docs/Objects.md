# Объекты в JavaScript

## Ссылочная природа объектов

Все объекты в JavaScript являются ссылочными типами данных. Это означает, что переменная хранит не сам объект, а ссылку на область памяти, где он находится.

## Пример того что все объекты ссылочные

> В примере видно что obj это просто ссылка на место в памяти и заменяя ссылку, мы никак не очищаем память

```js
let obj = { a: 1 };
let array = [obj];

obj = null;
console.log(array); // [ { a: 1 } ]
```
